April 8, 2015 by Jean in Learning

Getting started

This is the first post on this site! I wanted to work on a static site that is completely hand-coded. I'll probably transition to Jekyll at some point, so this site is a work-in-progress and will be see changes. While probably not the most elegant way to go about making at blog/portfolio site, this "from scratch" approach will be beneficial in getting down to the nitty gritty with respect to learning html/css/javascript and whatever framework eventually get implemented.

Today

Today, I started with Bootstrap's blog template, but ended up stripping away almost all of it in favor of styling preferences. Upon browsing online for a few blog templates, finally found a few I liked on Squarespace. I found that the Native template was my favorite for this project, so I looked at an example and then build this site around that. It is a skeleton at this point, and I'll continue to modify and layer on the dyanmic elements.

I edited the index.html and main.css files until I got this rendering. Building the DOM structure and all styling (initial "blog v2" commit) took a whole day for me.

Want to collaborate?

If you're completely new to front-end, this site might be a fun way to learn since everything is static. The site is hosted through github pages, so no costs involved. I'm only on first commit at time of this post, but go ahead and fork the repo if you like here.

Questions/Comments? Send me an email. I'm happy to chat.
