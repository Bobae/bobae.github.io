A new site

April 8, 2015 by Jean in Learning

Getting started

This is the first post on this site! I wanted to work on a static site that is completely hand-coded. I'll probably transition to Jekyll at some point, so this site will probably go through some iterations.

Today

Today, I started with Bootstrap's blog template, but eventually stripped away most of it in favor of my own styling preferences. Upon browsing online for a few blog templates, I found a few designs I liked on Squarespace. I liked the Native template design, so I looked at their example site and built this site around that style.

While this site is a functional prototype, I'm working on releasing another version in May with some additional features and refinement.
