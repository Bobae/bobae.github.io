↞ What's all this about now? ↠

April 8, 2015 by Jean in Learning

Getting started

This is the first post on this project! I wanted to work on a static site that is completely hand-coded. This site will be continually build-up. I'll probably transition to Jekyll at some point, but this project-based approach is probably the best for my learning.

Today

Today, I started with Bootstrap's blog template, but ended up stripping away almost all of it (the styling wasn't what I was looking for). Upon browsing online for a few blog templates, finally found a few I liked on Squarespace. I found that the Native template was my favorite for this project, so I looked at an example and then build this site around that. It is a skeleton at this point, and I'll continue to modify and layer on the dyanmic elements.

I edited the index.html and blog.css files until I got this rendering. Building the DOM structure and all styling (initial "blog v2" commit) took a whole day for me.
